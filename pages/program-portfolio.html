World of Warcraft Toolbox

Series of Excel add-ins developped in C# during the translation of World of Warcraft.

Text Rendering Tool

Features:

  • Converts custom tags in HTML tags and applies format for an easier review of the translation
  • Inserts game icons to check the final display
  • Creates gender and plural variants to spellcheck both versions of the texts

Error Report Tool

Features:

  • Detects automatically the selected language to check in multilanguage exports
  • Checks tokens match in source and target texts
  • Checks each category of lines complies with the style guide instructions (capitalization, typography, syntax, spacing, etc.)
  • Checks gender and plural token syntax and makes sure each kind of token is used in the right file
  • Highligts error when possible
  • Adds an error message for each type of error detected
